# MUgreroPV

## Juego de Peleas - Academia de Policía

Un juego de peleas 2D desarrollado en JavaScript puro para dos jugadores.

### Características

- Juego de peleas en 2D para dos jugadores
- Controles simples e intuitivos
- Sistema de salud y temporizador
- Mecánicas de salto y ataque
- Detección de colisiones
- Interfaz visual atractiva

### Cómo Jugar

1. Abre `index.html` en tu navegador web
2. Dos jugadores compiten en una batalla
3. El objetivo es reducir la salud del oponente a 0 antes de que se acabe el tiempo

### Controles

**Jugador 1:**
- W: Saltar
- A: Moverse a la izquierda
- D: Moverse a la derecha
- Espacio: Atacar

**Jugador 2:**
- Flecha arriba (↑): Saltar
- Flecha izquierda (←): Moverse a la izquierda
- Flecha derecha (→): Moverse a la derecha
- Enter: Atacar

### Tecnologías Utilizadas

- HTML5 Canvas para renderizado
- CSS3 para estilos
- JavaScript vanilla (sin frameworks)

### Instalación

No requiere instalación. Simplemente abre `index.html` en cualquier navegador moderno.
